Facebook Falafel

Facebook Falafel is a Facebook client for the terminal developed by K-Dog (Kevin Chavez) and Swayjay (Ajay Gandhi) at Big Red Hacks 2014. Simply put, Falafel is unproductively for the 1337.

How do I do?

First, make sure you have Node.js. Then:

$ npm install -g fb-falafel

Usage

To use Falafel, just type in facebook:

$ facebook

Use the --help parameter to get a list of options for facebook.
